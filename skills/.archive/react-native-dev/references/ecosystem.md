# React Native Ecosystem

## Navigation
| Library | Description |
|---------|-------------|
| [react-navigation](https://reactnavigation.org) | Standard navigation library (Stack, Tabs, Drawer) |
| [expo-router](https://docs.expo.dev/router/introduction) | File-based routing (Expo default) |

## UI Components
| Library | Description |
|---------|-------------|
| [lottie-react-native](https://github.com/lottie-react-native/lottie-react-native) | After Effects animations at runtime |
| [react-native-vector-icons](https://github.com/oblador/react-native-vector-icons) | Icon sets (Material, FontAwesome, Ionicons) |
| [react-native-maps](https://github.com/react-native-maps/react-native-maps) | Map component (Google Maps / Apple Maps) |
| [react-native-gifted-chat](https://github.com/FaridSafi/react-native-gifted-chat) | Chat UI component |
| [react-native-calendars](https://github.com/wix/react-native-calendars) | Calendar UI (Wix) |
| [react-native-paper](https://callstack.github.io/react-native-paper) | Material Design components |

## State Management
| Library | Description |
|---------|-------------|
| [zustand](https://github.com/pmndrs/zustand) | Minimal store with hooks (preferred) |
| [redux toolkit](https://redux-toolkit.js.org) | Predictable state container |
| [mobx](https://mobx.js.org) | Observable state |
| [tanstack-query](https://tanstack.com/query/latest) | Server state / async cache |

## Storage
| Library | Description |
|---------|-------------|
| [realm-js](https://www.mongodb.com/docs/realm/sdk/react-native) | MongoDB object database |
| [expo-sqlite](https://docs.expo.dev/versions/latest/sdk/sqlite) | SQLite for Expo apps |
| [mmkv](https://github.com/mrousavy/react-native-mmkv) | Fast synchronous key-value storage |
| [watermelondb](https://watermelondb.dev) | High-performance reactive database |

## Authentication
| Library | Description |
|---------|-------------|
| [expo-auth-session](https://docs.expo.dev/guides/authentication) | OAuth flow (Google, GitHub, etc.) |
| [firebase auth](https://firebase.google.com/docs/auth) | Full auth solution |
| [clerk](https://clerk.com/docs) | Drop-in auth |
| [cognito](https://aws.amazon.com/cognito) | AWS auth service |

## Testing & CI/CD
| Tool | Description |
|------|-------------|
| [detox](https://wix.github.io/Detox) | Gray-box E2E testing (Wix) |
| [maestro](https://maestro.mobile.dev) | Mobile E2E testing framework |
| [fastlane](https://fastlane.tools) | Automate beta deployments |
| [eas build](https://docs.expo.dev/build) | Expo cloud builds |
| [appcircle](https://appcircle.io) | Mobile CI/CD platform |

## Performance
- [react-native-reanimated](https://docs.swmansion.com/react-native-reanimated) — 60fps animations on UI thread
- [hermes](https://hermesengine.dev) — JS engine optimized for RN (default Expo SDK 50+)
- [flipper](https://fbflipper.com) — Mobile app debugger
- [react-native-skia](https://shopify.github.io/react-native-skia) — 2D graphics via Skia
