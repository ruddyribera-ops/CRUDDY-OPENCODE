# Frontend Ecosystem & Libraries

## Component Libraries
| Library | Description |
|---------|-------------|
| [shadcn/ui](https://ui.shadcn.com) | Copy-paste component library built on Radix + Tailwind |
| [ant-design](https://ant.design) | Enterprise UI design system |
| [material-ui](https://mui.com) | Material Design React components |
| [chakra-ui](https://chakra-ui.com) | Modular, accessible component library |
| [mantine](https://mantine.dev) | 100+ hooks & components, dark mode built-in |
| [headlessui](https://headlessui.com) | Unstyled accessible components |
| [radix-ui](https://radix-ui.com) | Unstyled primitive components (shadcn/ui base) |

## State Management
| Library | Pattern |
|---------|---------|
| [zustand](https://github.com/pmndrs/zustand) | Minimal store with hooks (preferred) |
| [jotai](https://github.com/pmndrs/jotai) | Atomic state (lighter Recoil) |
| [redux toolkit](https://redux-toolkit.js.org) | Predictable state container |
| [mobx](https://mobx.js.org) | Observable state |
| [tanstack-query](https://tanstack.com/query) | Server state / async cache |
| [swr](https://swr.vercel.app) | Stale-while-revalidate data fetching |
