# Vision Analysis Ecosystem

## Core Libraries
- opencv-python — image/video processing, feature detection
- Pillow (PIL fork) — image open/save/resize/crop
- scikit-image — scientific image processing
- torchvision — PyTorch vision transforms, model zoo
- albumentations — fast GPU-friendly augmentation
- imgaug — stochastic augmentation

## Object Detection
- YOLOv8 (Ultralytics) — real-time, 100+ pre-trained models
- DETR (HuggingFace) — end-to-end detection with transformers
- Grounding-DINO — open-set detection with text prompts

## Face & Expression
- DeepFace — face recognition + age/gender/emotion
- face-recognition (dlib wrapper)
- MediaPipe — lightweight face mesh
- InsightFace — 2D/3D face analysis

## Image Generation
- diffusers (HuggingFace) — stable diffusion pipeline, LoRA, ControlNet
- ComfyUI — node-based stable diffusion workflow
- InvokeAI — streamlined stable diffusion

## OCR (also see ocr-tools skill)
- EasyOCR — 80+ languages, deep learning
- PaddleOCR — 80+ languages, high accuracy
- Tesseract — classic OCR, 100+ languages
- Surya — transformer-based OCR

## Video Processing
- ffmpeg-python — FFmpeg wrapper
- Decord — fast video decoder (GPU accelerated)
- MoviePy — video editing, compositing
- PySceneDetect — scene change detection

## Classification & Embedding
- timm (HuggingFace) — 800+ pre-trained image models
- CLIP (OpenAI) — zero-shot classification
- OpenCLIP — open-source CLIP implementations
- Vision Transformer (ViT) — attention-based classification
