# Flutter Ecosystem (awesome-flutter)

## UI Component Libraries
| Library | Description |
|---------|-------------|
| [shadcn-flutter](https://github.com/nicholasptran/shadcn_flutter) | Copy-paste Tailwind-style components for Flutter |
| [forui](https://forui.dev) | Minimalist widget library (Radix-inspired) |
| [tdesign-flutter](https://github.com/Tencent/tdesign-flutter) | Tencent enterprise design system |
| [material 3](https://m3.material.io) | Material Design 3 (built-in) |

## Charts
| Library | Description |
|---------|-------------|
| [fl_chart](https://github.com/imaNNeoFighT/fl_chart) | Interactive charts (line, bar, pie, radar) |
| [graphic](https://github.com/entronad/graphic) | Grammar of Graphics declarative charts |
| [bezier-chart](https://github.com/aeyrium/bezier-chart) | Beautiful bezier curve charts |

## State Management
| Library | Pattern |
|---------|---------|
| [riverpod](https://riverpod.dev) | Compile-safe provider |
| [bloc](https://bloclibrary.dev) | Event-driven state |
| [provider](https://pub.dev/packages/provider) | Simple DI (Riverpod predecessor) |
| [get](https://pub.dev/packages/get) | All-in-one (state, DI, routes) |
| [flutter_bloc](https://pub.dev/packages/flutter_bloc) | Bloc UI integration |

## Storage
| Library | Description |
|---------|-------------|
| [hive](https://pub.dev/packages/hive) | Lightweight key-value storage |
| [sqflite](https://pub.dev/packages/sqflite) | SQLite for Flutter |
| [isar](https://isar.dev) | High-performance NoSQL database (Hive successor) |
| [drift](https://drift.simonbinder.eu) | Reactive persistence library |
